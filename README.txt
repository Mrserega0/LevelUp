2D платформер игра по типу Level Devil.
Использую библиотеку SFML. И так же рисование уровней через Tiled.
Требования к компиляции.
1) SFML 2.6.2
2) C++ 17
Управление:
A/D - движение
SPACE - прыжок
R - Респавн при смерти
ESC - меню уровней
Основная механика работа триггеров через Tiled
action {string} что должен выполнить триггер
chain {int} - выбирается главный триггер например chain 1 и его триггеры которые могут иметь chain 1 выполняясь тогда когда игрок вошел в главный триггер
delay {float} - задержка перед выполнением триггера
tile {int} - выбирает локальный ID тайлсета, используется в spawn_block
1) action.spawn_spike
2) action.spawn_block
3) action.remove_block
4) action.remove_spikes
5) action.chain_start - запускает по номеру chain триггеры
